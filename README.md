# Polaris

Polaris is a PHP framework designed to simplify the development of robust and scalable web applications. This framework includes an ORM, models for MySQL, migration control, SQL cache management with Redis, MySQL version control via console, a template engine, and a router.

## Features

- **ORM (Object-Relational Mapping):** Easily interact with MySQL databases.
- **Models for MySQL:** Simplifies the creation and management of data models.
- **Migration Control:** Manage database schema changes effortlessly.
- **SQL Cache Management with Redis:** Improve performance with Redis caching for SQL queries.
- **MySQL Version Control via Console:** Track and manage database versions from the console.
- **Template Engine:** Build dynamic and reusable templates.
- **Router:** Define and handle application routes effectively.
